duppy
=====

Duppy - is a simple image duplicates finder using [Perceptual Hash Algorithm](http://www.hackerfactor.com/blog/index.php?/archives/432-Looks-Like-It.html) written in Python.

Requirements:

 * Python 2.6.x and above
 * PIL (Python Image Library) 1.1.7

How to run:
```bash
# python duppy.py <dir_name>
```

For demo:
```bash
# python duppy.py test_images\
```
